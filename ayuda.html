Sección de ayuda

Juegos

Sección que agrupa los distintos juegos y utilidades interactivas desarrolladas en el proyecto MotoGP Desktop.

Juego de memoria con cartas

Juego interactivo en el que el usuario debe emparejar cartas iguales utilizando la memoria visual. Se ha implementado mediante programación orientada a objetos en JavaScript y gestión de eventos del DOM.

Cronómetro (JavaScript)

Herramienta diseñada para medir el tiempo transcurrido durante las pruebas. Se controla mediante tres botones (Arrancar, Parar y Reiniciar) y muestra el tiempo en minutos, segundos y décimas.

Está implementado en ECMAScript mediante una clase orientada a objetos denominada Cronometro, utilizando el objeto Temporal cuando está disponible, con alternativa basada en Date.

Cronómetro (PHP)

Versión del cronómetro implementada en PHP, utilizada en el contexto del servidor para controlar y registrar tiempos asociados a pruebas. Permite iniciar, detener y obtener el tiempo medido, facilitando su almacenamiento en base de datos junto con los resultados del test.

Configuración Test

Apartado destinado a la preparación del entorno de pruebas en el servidor y a la gestión de la base de datos utilizada por el test de usabilidad (por ejemplo, desde el panel de acciones disponible en test.php).

  • Inicializar la base de datos del test.
  • Reiniciar los datos almacenados.
  • Eliminar completamente la información de las pruebas.
  • Exportar los resultados para su análisis posterior.

Pruebas de usabilidad

Sección dedicada a la realización de pruebas de usabilidad sobre el sitio. El usuario introduce sus datos de perfil, responde a un cuestionario y se registra el tiempo empleado, además de comentarios y valoración final.

Los datos recogidos se almacenan en la base de datos para su análisis posterior, permitiendo justificar mejoras de usabilidad en la documentación del proyecto.
